# Joe_Japan_Tourism

I worked on this project over the Spring of 2026. I wanted to practice working with time-series data in a context that I was interested in. I ended up using data of international visitor arrivals to Japan. I created an ETL pipline to get the data from www.tourism.jp, then analyzed the results in both R and Python. 

The following two PDFs contain my analysis of part visitor data and forecasts for international visitor levels to Japan over the next few years.  

[Part 1: Exploring the Data](Japan_Tourism_Analysis_Pt1.pdf)
[Part 2: Creating Forecasts](Japan_Tourism_Analysis_Pt2.pdf)
